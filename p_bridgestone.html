Bridgestone

Atomic Design System

A tokenized design system connecting Figma components to React components via Zeplin and Storybook.

Context

My team was tasked with creating the design system for a new Bridgestone B2B offering. Bridgestone was in the early stages of rolling out new brand guidelines for print and broadcast materials. We translated these guidelines into a user interface for professionals to use as a part of their work day.

Structure

Naming conventions were chosen with front-end development in mind, and component structures were designed for creating prototypes in a fast-paced environment.

Documentation was kept at the juncture between design and development for easy access by both teams.

ADA compliance was built in from the ground up. Keyboard navigation is not an afterthought but is envisioned as a way for professionals to use the software as efficiently as possible.

A design system is a living system that must continuously adapt. Without a prescribed system of governance, it can quickly fall apart.
